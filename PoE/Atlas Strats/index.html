    Alch n Go Betrayal/Settlers/Eagon Rush 3-in-1¶

    • Credit: bwcloud & epsi
    • PoE Planner: PoE Planner
    • Scarabs: - (optional: Reinforcements)
    • Map Device: Settlers (5c)
    • Altar: Eater (Divine Shrine Fishing) oder nach Präferenz
    • Anleitung:
    • Alch n Go T16 / 16.5, hoher Durchsatz (1-3Min pro Map)
    • Aufbau Betrayal Board, viele Konkurrenten oder Verbündeten, Fokus auf Füllung von Intelligence für Safehouse Spam
    • Memory Tears anklicken und bei "Reverence" gutes extra Einkommen, Betrayal kann auf Eagon Missionen genau so weiter gemacht werden (Strat in der Strat)
    • Settlers: Svalinn-Crater Hunting, Lost Shipments für raw Currency, teure Tattoos etc
    • Investition: 0-1c für den Scarab, sehr günstig, hohe Menge ist das Ziel, High Value Drops: Safehouses Allflames für Scarab oder Goldfarm, Syndicate Medallion (Katarina Fight, potentielle Big Tickets oder direkter Sale), Svalinn-Potential durch viele Crater dank Settlers, High Value Tattoos und Raw Currency durch Settlers.

    Woche 1¶

    ==========

    Maven Deli Mirror Guardian Rotas¶

    • Credit: Path of Evening (found by epsi)
    • Video: https://youtu.be/jTy-LE5taPY
    • PoE Planner: PoE Planner
    • Scarabs: Delirium, Mania x2, Neuroses, Conversion oder Paranoia
    • Search Link Maps: Trade - Path of Exile
    • Map Device: Quant (free)
    • Altar: Maven (Witnessing der Bosse für Conquerer Invitation + normale Invitation dank verschiedener Maps)
    • Anleitung:
    • Conquerer Maps (Baran, Drox, Veritana, Al-Hezmin) mit min 100% Quant laufen
    • Delirium Mirror am Start nehmen, zum Boss rushen, Conquerer killen, Deli beenden.
    • Maven Invitiations auf 85+ Quant rollen
    • Investition: 1c Scarabs, Maps relativ günstig, stetige gute Profite weil div. Mischung von Einnahmequellen (Maven Invitations, Delirium Splinter, Raw Currency, Cluster Jewels, div. Conquerer Orbs), teilweise harte Maps aber overall eher "easy going" je nach Build

    Boxes T16.5/T17 Strongboxes "Small Version"¶

    • Credit: Gimmi
    • Video: https://youtu.be/jTy-LE5taPY
    • PoE Planner: PoE Planner
    • Scarabs: 1x Discernment, 1x Potency, 3x Ambush
    • Search Link Maps: Trade - Path of Exile
    • Map Device: Ambush (4c)
    • Altar: Eater (Divine Shrines, Eldritch Minions Quant etc)
    • Anleitung:
    • T16.5 oder T17 auf viel "Currency" (100+) und "Scarabs" (70+), sowie Packsize (40+) rollen
    • Strongboxen öffnen und Monster killen. Gute Strongboxen: Arcanist, Diviner, Ornate - Wichtig in T17: Boss killen für Fragmente, beste Maps: Abomination, Ziggurat, Sanctuary (Uber Maven/Eater Fragmente)
    • Investition: Mittlerer Invest, Maps ggf. teurer, T17 je nach Map teurer, lohnt sich aber immer, idR. wird der Invest "verdoppelt" mit Chancen auf Big Tickets durch Divination Cards, Double Divines, Double Valdos etc.

    Boxes T17 Strongboxes "Big Version"¶

    • Credit: Gimmi
    • Video: https://youtu.be/jTy-LE5taPY
    • PoE Planner: PoE Planner
    • Scarabs: 1x Discernment, 1x Potency, 1x Ambush, 1x Containment, 1x Hidden Compartments (einmal jeden Ambush Scarab!)
    • Map Device: Ambush (4c)
    • Altar: Eater (Divine Shrines, Eldritch Minions Quant etc)
    • Search Link Maps: Trade - Path of Exile
    • Anleitung:
    • T17 auf viel "Currency" (100+) und "Scarabs" (70+), sowie Packsize (40+) rollen
    • Strongboxen öffnen und Monster killen. Gute Strongboxen: Arcanist, Diviner, Ornate - Containment: Alle Packs sind in Strongboxen eingesperrt, 30+ Strongboxen öffnen. Wichtig in T17: Boss killen für Fragmente, beste Maps: Abomination, Ziggurat, Sanctuary (Uber Maven/Eater Fragmente)
    • Investition: Hoher Invest, Containment Scarab oft 1-2 Div pro Stück, Maps ggf. teurer, T17 je nach Map teurer, zahlt sich aber aus da 30-40 Strongboxen, öfter "an der Slotmachine ziehen" - idR. wird der Invest "verdoppelt" mit Chancen auf Big Tickets durch Divination Cards, Double Divines, Double Valdos etc.
